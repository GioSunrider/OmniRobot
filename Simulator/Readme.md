1. Instalar Processing:

  Descarga Processing desde https://processing.org/download
  Instálalo en tu computadora

2. Ejecutar el simulador:

  Copia el código del simulador en un nuevo sketch de Processing
  Guarda el sketch con nombre "Robot3W_Simulador"
  Haz clic en el botón "Run" (triángulo)

3. Probar controles:

  Usa WASD para mover el robot
  Q/E para rotar
  ESPACIO para activar motores
  X para desactivar
  L para láser
  R para reset
